# コーディング規約（開発）

> 架空のサンプルデータです。

## 共通
- 命名は英語、説明的に（略語の乱用を避ける）
- 1関数1責務、副作用を最小化
- 例外は握りつぶさずログ化

## Python
- `datetime.utcnow()` は使用禁止。`datetime.now(timezone.utc)` を使用
- フォーマッタ/リンタ: ruff
- 型ヒントを付与

## レビュー
- PR は小さく保つ（目安 400 行以内）
- 最低1名のレビュー承認が必要
- CI（lint/test）グリーンが必須

## セキュリティ
- 機密情報をコード・ログに残さない
- 依存はバージョン固定
