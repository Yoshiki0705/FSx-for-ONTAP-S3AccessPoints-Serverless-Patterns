# UC4: Media VFX — テストデータ

## 概要

メディア VFX レンダリング品質チェックワークフローのテスト用サンプルデータ。

## ファイル構成

```
test-data/media-vfx/
└── renders/
    └── sample-render-job-001.json   # レンダリングジョブメタデータサンプル
```

## サンプルデータの説明

### sample-render-job-001.json

レンダリング出力の品質チェック結果サンプル。以下を含む:
- レンダリングメタデータ（プロジェクト、ショット、フレーム、解像度、エンジン）
- 品質チェック結果（黒フレーム、ファイアフライ、ノイズレベル、色クリッピング）

## 使用方法

1. 品質チェック Lambda のユニットテストで使用
2. CloudFront 配信パイプラインのテストデータとして使用
3. Bedrock レポート生成のインプットとして使用
